How to Use
Step-by-step guide to get started
1
Power on NeuroGrid

Connect the ESP32 board to a power source via USB or battery. Wait for the onboard LED to stabilize — this means it's ready.

2
Connect to the Wi-Fi hotspot

On your phone or laptop, open Wi-Fi settings and connect to the network named NeuroGrid-AP. No password is required by default.

3
Open the control panel

Open any browser and navigate to 192.168.4.1. This page will load — you're now in control.

4
Select a mode

Use the Mode Control section above to choose an operating mode. Tap the desired mode card, then press Apply Mode.

5
Done!

The device will apply your selected mode immediately. The status bar confirms what mode is active. You can switch modes anytime.

User Manual
Technical reference and safety information

Overview

NeuroGrid is an ESP32-based smart control device designed for [your use case here]. It creates its own local Wi-Fi access point and serves this web interface, allowing real-time configuration without any external infrastructure.

Operating Modes

Mode 1 — The default operating configuration. Suitable for standard use cases. The device runs at baseline parameters and conserves power.

Mode 2 — An alternate mode for advanced scenarios. Modify this description once you finalize the mode behavior.

Mode 3 — A specialized configuration for targeted applications. Update with your specific mode details.

Network Information

The ESP32 operates as a Wi-Fi Access Point (AP mode). It does not connect to any external router. The default IP address is 192.168.4.1 and the SSID is NeuroGrid-AP. These can be modified in firmware if needed.

Troubleshooting

If the control panel doesn't load, ensure you are connected to the NeuroGrid Wi-Fi (not your home/office network). If the device is unresponsive, try power cycling it. Allow 5–10 seconds after boot before connecting.

⚠️ Do not operate NeuroGrid near high-moisture or high-heat environments. Ensure stable power supply during mode switching to avoid unexpected resets.
